Home work for go courses 01-05.02.2021

Для запуска программы ввести команды:

1. Напишите генератор квадратов натуральных чисел. Найдите самую длинную последовательность нулей в двоичном представлении целого числа. 
   
   go run ./cmd/first/main.go
   
2. Вращение массива. Дан массив целых чисел, необходимо посчитать, количество уникальных значений.

   go run ./cmd/second/main.go
   
3. Написать HTTP эхо сервис. На любой POST запрос он должен возвращать тело запроса как ответ.
   
   go run ./cmd/third/main.go
   
   Запросы посылать на localhost:8080/post
